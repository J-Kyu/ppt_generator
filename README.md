# PPT Generator

## Description
This project is an offline application using Flet, designed to analyze the layout of PPTX files and dynamically generate input forms, allowing users to create new presentations.

## Tech Stack
- Python 3.13
- Package Manager: `uv`
- GUI: `flet`
- PPT Manipulation: `python-pptx`
- Data: `pydantic`
- Packaging: `pyinstaller`

## Setup and Run
```bash
uv run src/main.py
```
